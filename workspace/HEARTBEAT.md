# HEARTBEAT.md

## Learnings

On each heartbeat, review recent activity and update `MEMORY.md` with anything worth remembering:

- New services discovered in the Tempo MPP directory and their pricing
- Services that were unreliable or slow
- Patterns in what the user asks for most
- Price changes on frequently used services
- Wallet balance — notify the user if it's running low

Keep entries concise. The goal is a running log that makes future recommendations faster and smarter.
